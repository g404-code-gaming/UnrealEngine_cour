# Utiliser une vidéo comme texture 

Dans vos projets Unreal Engine, vous pourrez utiliser des vidéos MP4 en guise de texture pour vos objets.

Créer les dossiers 

Créé le Media source 

Ouvrir le dossier pour y mettre la vidéo 

dans le media source, mettre le chemin de la vidéo

créé le media player 

dedans, mettre la vidéo (loop + play on open)

appliquer la texture sur l'objet pour créer le material 

dans le levelblueprint, créer l'évènement qui lance l'animation
